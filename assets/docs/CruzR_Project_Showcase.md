# CruzR – Project Description

## Project

**CruzR – Humanoid Service Robot Platform**

### Overview

An Android-based application framework developed for the **CruzR humanoid service robot**.

The project originally started as my **Master's thesis** at Óbuda University, where I designed and implemented a reusable Android application framework for the CruzR robot platform. The solution was later adopted in an industrial environment and further maintained and extended as part of a successful government-funded project.

The platform runs on an Android/AOSP and Ubuntu-based service robot and provides application-level functionality for customer-specific robotic solutions. My responsibilities included application architecture, feature development, customer-specific customization, functional testing, on-site deployment, troubleshooting, and long-term maintenance.

Rather than being a simple mobile application, the project required combining Android software development with robotics, AI integration, hardware interaction, and real-world system deployment.
